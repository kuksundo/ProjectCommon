What is TFormAbout?
===================
About dialog form with license text and splash screen component.

Features:

Splash mode or About mode with license text
Alpha blended close
Background picture logo
Displays links 3-3 with copyright and "Registered to..." message
Check for updates on internet


Requirements:
=============
Delphi


Installation:
======================
Unpack the TFormAbout package (TFormAbout X.X.zip).
In Delphi menu: Component: Install component
Unit file name: select "FormAboutDefs.pas"
Compile and save package.
If everything went smooth you can access TFormAbout on 3delite's tab.
